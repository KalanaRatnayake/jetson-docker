## jetson-ros-pytorch:humble-r32.7.1

### Pull or Build

Pull the docker container
```bash
docker pull ghcr.io/kalanaratnayake/jetson-ros-pytorch:humble-r32.7.1
```

Build the docker container
```bash
docker buildx build --load --platform linux/arm64 -f ros/humble/r3271.Dockerfile -t jetson-ros-pytorch:humble-r32.7.1 .
```

### Start

Start the docker container

```bash
docker run --rm -it --runtime nvidia --network host --gpus all -e DISPLAY ghcr.io/kalanaratnayake/jetson-ros-pytorch:humble-r32.7.1 bash
```

### Test

Run the following commands inside the docker container to confirm that the container is working properly.
```bash
python3 -c "import torch; print(torch.__version__)"
python3 -c "import torchvision; print(torchvision.__version__)"
```

Run the following commands inside the docker container to confirm that the container is working properly.
```bash
ros2 run demo_nodes_cpp talker
```

Run the following commands on another instance of ros container or another Computer/Jetson device installed with ROS humble to check 
connectivity over host network and discoverability (while the above command is running).
```bash
ros2 run demo_nodes_py listener
```
