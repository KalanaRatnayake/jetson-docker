## l4t-cuda:12.6.11-runtime

- CUDA 12.6 related runtime libraries

### Pull or Build

Pull the docker container
```bash
docker pull nvcr.io/nvidia/l4t-cuda:12.6.11-runtime
```

<br>

### Start

Start the docker container
```bash
docker run --rm -it --runtime nvidia --network host --gpus all -e DISPLAY nvcr.io/nvidia/l4t-cuda:12.6.11-runtime bash
```
