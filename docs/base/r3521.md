## l4t-base:r35.2.1

### Pull or Build

Pull the docker container
```bash
docker pull nvcr.io/nvidia/l4t-base:r35.2.1
```

<br>

### Start

Start the docker container
```bash
docker run --rm -it --runtime nvidia --network host --gpus all -e DISPLAY nvcr.io/nvidia/l4t-base:r35.2.1 bash
```

<br>

### Test

Start the docker container
```bash
docker run --rm -it --runtime nvidia --network host --gpus all -e DISPLAY nvcr.io/nvidia/l4t-base:r35.2.1 bash
```

And in the internal terminal run,
